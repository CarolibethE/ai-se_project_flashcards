# FLASHCARD APP

My first project in TripleTen's AI-Assisted Software
Engineering program. It includes decks of flashcards,
each of which can be viewed in a carousel.

## Features

Home View
- Displays all flashcard decks in a responsive grid layout.
- Shows the deck name and number of cards.
- Allows decks to be deleted.

Open Deck View
- Displays all cards belonging to a selected deck.
- Allows users to flip cards to view answers.
- Allows cards to be deleted.
- Includes a Practice button that opens the carousel study mode.

Carousel Practice View
- Displays one flashcard at a time.
- Navigate between cards using previous and next buttons.
- Flip cards to reveal answers.
- Navigation buttons are disabled when the beginning or end of the deck is reached.

Responsive Design
- Layout adapts to mobile, tablet, and desktop screen sizes.
- Card grids resize appropriately on smaller screens.
- Mobile action buttons remain fixed at the bottom of the screen.
- Carousel layout adjusts for smaller viewports.

## Technologies Used

- HTML
- CSS
- Flexbox
- Responsive Design
